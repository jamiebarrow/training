Snake

This project is about creating a snake game clone.

The rules of the game are below:

  • The player controls a snake which starts off at a random position on the map
  • The map is divided into squares, and the snake initially is 3 squares in length
  • At each tick of the game, the snake moves one square forward in the direction it is moving
  • You can change the direction of the snake using the direction keys
  • Apples appear on the map, and if the snake eats an apple it will grow in length by one square
  • If a snake moves into a wall, or the edge of the map, the snake dies
